# Currency-Converter
A simple currency converter web app that converts one currency to another using the free fawazahmed0 Exchange API .



# About.

This is a currency converter web app built using HTML, CSS, and JavaScript.
It fetches real-time exchange rates from the fawazahmed0 Exchange API
 (a free and open-source API) to convert between different world currencies instantly.

 # Features:

Convert from one currency to another in real-time.

Supports 150+ world currencies.

Simple and fast API integration (no API key required).

Option to show latest or historical rates.

Responsive and lightweight design.

# Technologies Used:

HTML

CSS

JavaScript (Fetch API)

fawazahmed0 Exchange API
